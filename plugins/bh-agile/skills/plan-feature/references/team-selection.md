## Agent Roles

### UX/Design Focus Agents

| Role | Expertise | Used in |
|------|-----------|---------|
| **Product Designer** | User flows, screens, interaction patterns, error UX, accessibility | Full-stack (all), UX/Design (all) |
| **UI Expert** | Component hierarchy, state management, responsive design, design system | Full-stack (all), UX/Design (all) |
| **Target Persona** | User perspective, usability feedback, competitive comparison | Full-stack (Complex), UX/Design (Medium/Complex) |

### Architecture Focus Agents

| Role | Expertise | Used in |
|------|-----------|---------|
| **Software Architect** | System design, data model, integration points, technology choices | Architecture (all), Backend only (all) |
| **API Designer** | Endpoints, request/response schemas, error handling, versioning | Backend only (all) |
| **Security Analyst** | Threat model, auth, data protection, input validation, compliance | Architecture (Medium/Complex), Backend only (Complex) |
| **Performance Engineer** | Performance budgets, bottleneck analysis, caching, monitoring | Architecture (Complex) |

### Cross-Focus Agent

| Role | Expertise | Used in |
|------|-----------|---------|
| **Domain Expert** | Business rules, industry best practices, regulatory constraints, terminology | Full-stack (Medium/Complex), UX/Design (Complex), Architecture (all), Backend only (Medium/Complex) |
