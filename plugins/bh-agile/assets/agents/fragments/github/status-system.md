Status is tracked via labels. Issues are `open` or `closed` (GitHub state).

| Status | Label | GitHub State |
|--------|-------|--------------|
| Backlog | `status:backlog` | open |
| In Progress | `status:in-progress` | open |
| Done | _(no status label)_ | closed |

When moving to "Done", close the issue and remove any status label.
When moving to "In Progress", add `status:in-progress` and remove `status:backlog`.
