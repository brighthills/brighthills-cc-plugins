## Adding New Feature Documentation

1. Create folder: `docs/features/[feature-name]/`
2. Add `README.md` with overview
3. Add numbered docs for complex features (00-ARCHITECTURE.md, 01-IMPLEMENTATION.md, etc.)
4. Update this index
5. When complete: move folder to `docs/features/COMPLETED/` and move entry to Completed table
